📖 Getting Started Guide

Everything you need to know to start using AO3 Tracker Bot effectively.

⌨️ Keyboard Shortcut

Press Ctrl + K (or Cmd + K on Mac) to jump directly to the command search!

🚀 Adding Works to Your Library

1

Automatic Detection (Easiest!)

Just paste any AO3 link in a channel where the bot is present. The bot automatically detects the link and adds the work to your library. No command needed!

https://archiveofourown.org/works/12345678

The bot will:

  • React with ✅ emoji when successful
  • Display an embed with all the work's information
  • Automatically save it to YOUR personal library
🤖 Multiple Links

The bot detects ALL AO3 links in a single message! Post multiple links at once and they'll all be added.

2

Using Commands

You can also use /workinfo with a work ID or URL to add and view a work:

/workinfo work_id:12345678
/workinfo work_id:https://archiveofourown.org/works/12345678
💡 First Load Takes Time

The first time you add a work, the bot needs to fetch data from AO3 (takes 1-3 minutes). After that, it's instant because data is cached!

🔒 Restricted Works

If a work is restricted (requires AO3 login), the bot can't fetch it automatically. But you can still add it!

  1. Bot shows error message with clickable button
  2. Click "📝 Show How to Add Manually"
  3. Follow detailed instructions to use /quickadd
  4. The work ID is pre-filled for convenience!

📊 Tracking Your Reading

Reading Status

Every work can have one of three statuses:

📕 Unread - Haven't started yet
📖 Reading - Currently reading
📗 Completed - Finished reading

Change status with:

/markreading work_id:12345678    # Currently reading
/markread work_id:12345678       # Finished
/markunread work_id:12345678     # Reset to unread

Progress Tracking

Track exactly how far you've read:

/progress work_id:12345678 chapters_read:15    # Read 15 chapters
/addchapter work_id:12345678 chapters:3        # Add 3 more chapters
/showprogress work_id:12345678                 # See visual progress bar
✨ Auto-Complete Magic

When your progress reaches 100%, the bot automatically:

  • Changes status from 📖 Reading → 📗 Completed
  • Records the finish date
  • Sets chapters_read = total chapters

Example: A 10-chapter fic becomes "completed" when you use /progress work_id:12345678 chapters_read:10

Rating & Notes

Add your personal rating and thoughts:

/rate work_id:12345678 rating:5                                    # 5 stars!
/note work_id:12345678 note_text:Amazing slow burn, cried at the end
/tag work_id:12345678 tags:comfort fic, slow burn, favorite

📚 Organizing with Lists

Create custom lists to organize your library however you want:

/addlist list_name:favorites work_id:12345678      # Add to "favorites" list
/addlist list_name:to-reread work_id:12345678      # Add to "to-reread" list
/showlist list_name:favorites                       # View all works in list
/mylists                                            # See all your lists

List Ideas

  • favorites - Your all-time best
  • comfort reads - For bad days
  • to-reread - Works worth revisiting
  • currently following - WIPs you're tracking
  • rec'd by friends - Recommendations from others

Making Lists Public

Share your lists with others:

/makepublic list_name:favorites                                    # Make list public
/listdesc list_name:favorites description:My all-time favorite fics!   # Add description

Others can then view and subscribe to your list!

🔍 Finding Works

Search Your Library

/search keyword:coffee shop           # Search all fields
/fandom name:Harry Potter             # Filter by fandom
/author name:AuthorName               # Filter by author
/series name:SeriesName               # Filter by series
/tagged tag:comfort fic               # Search your custom tags

Browse by Status

/unread              # To-read pile
/reading             # Currently reading
/read                # Finished works
/completed           # Complete fics (not WIPs)
/wip                 # Works in progress
/toprated            # Your highest rated

👥 Social Features

Recommending to Friends

/recommend user:@Friend work_id:12345678 note:You'll love this one!

Your friend will receive a notification and can accept or decline:

/recs                         # View pending recommendations
/acceptrec work_id:12345678   # Accept and add to library
/declinerec work_id:12345678  # Decline

Public Lists & Subscriptions

/publiclists                              # Browse all public lists
/viewlist user:@User list_name:favorites  # View someone's list
/subscribe user:@User list_name:favorites # Get notified of updates
📢 When You Get Notified

Subscribers receive DM notifications when:

  • ✅ New work is added to a list you're subscribed to
  • ❌ NOT when works are removed
  • ❌ NOT when list description changes

Book Clubs

Create or join reading groups:

/clubs                                        # See all clubs
/joinclub club_name:ClubName                  # Join a club
/createclub club_name:MyClub                  # Create your own
/setclubread club_name:MyClub work_id:12345678  # Set the current read (owner)
📬 Club Notifications

Members receive DM notifications when the owner sets a new club read. The notification includes:

  • Work title, author, and link
  • Instructions to use /acceptclubread to add it to your library

Toggle with: /clubnotify setting:off

📊 Statistics & Graphs

Visualize your reading habits:

/stats               # Text statistics overview
/readinggraph        # Library growth over time
/fandomgraph         # Top fandoms pie chart
/ratingsgraph        # Your rating distribution
/allgraphs           # Generate ALL graphs at once!

View someone else's graphs:

/readinggraph user:@FriendName    # View friend's library growth
📈 Graph Types Available
  • readinggraph - Library growth over time (line)
  • wordsgraph - Cumulative words (line)
  • fandomgraph - Top fandoms (pie chart)
  • statusgraph - Reading status breakdown (bar)
  • ratingsgraph - Rating distribution (bar)
  • monthlygraph - Works added per month (bar)
  • servergraph - Compare with server members (horizontal bar)

💾 Your Data

Auto-Save

Your data automatically saves every 5 minutes and whenever the bot shuts down. You never need to manually save.

⚠️ Important: Data Saving
  • ✅ Auto-saves every 5 minutes
  • ✅ Saves on bot restart/shutdown
  • ⚠️ If bot crashes unexpectedly, last 5 min may be lost
  • 💡 Use /export regularly for backups!

Export Backup

/export

Downloads a JSON file with all your data. Great for backups!

💾 What's Included in Export?

Included:

  • All works with full metadata
  • Your ratings, notes, tags, progress
  • All reading lists and work assignments

NOT included:

  • Recommendations (sent/received)
  • Public list subscribers
  • Book club memberships
  • Settings

Filename format: ao3_export_YourName_20240115_143022.json

Privacy

Your library is completely private. Other users cannot see your works, ratings, notes, or progress unless you:

  • Make a list public
  • Send a recommendation
  • Share your stats (they can use /userstats user:@You)

⚙️ Settings

Customize how the bot works for you:

/settings                           # View all your current settings
/recnotify setting:dm               # Receive recs via DM only
/clubnotify setting:off             # Disable club notifications
/subnotify setting:off              # Disable subscription notifications
/listlimit limit:15                 # Show 15 items in lists (default: 10)

Default Settings

Setting Default Value Command to Change
Recommendation Notifications 📩💬 Both (DM + Channel) /recnotify
Profile Privacy 🌐 Public /privacy
Display Mode 📖 Detailed /displaymode
List Display Limit 10 items /listlimit
Club Notifications ✅ Enabled /clubnotify
Subscription Notifications ✅ Enabled /subnotify

📝 Quick Add (Manual Entry)

When the bot can't fetch a work automatically (restricted/private works), you can add it manually:

/quickadd work_id:12345678 title:"The Best Fic Ever" author:"AuthorName" fandom:"Marvel"

Required Parameters

Parameter Description Example
work_id AO3 work ID or URL 12345678
title Work title "The Best Fic Ever"
author Author name(s), comma separated "Author1, Author2"
fandom Fandom name(s), comma separated "Marvel, MCU"

Optional Parameters

Parameter Description Options/Example
rating Content rating General, Teen, Mature, Explicit, Not Rated
category Category M/M, F/F, F/M, Gen, Multi, Other
status Completion status Complete, WIP, Unknown
words Word count 50000 or 50,000
chapters Chapter count 10/10 or 5/?
relationships Ship tags, comma separated "Steve/Bucky, Tony/Pepper"
characters Character names, comma separated "Steve Rogers, Bucky Barnes"
tags Freeform tags, comma separated "Fluff, Angst, Hurt/Comfort"
summary Work summary "A story about..."
series Series name(s), comma separated "My Series Name"
💡 Pro Tip

Use quotes around values that contain spaces! For example: title:"My Amazing Fic"

ℹ️ Good to Know

🔗 Interactive Links

Every piece of metadata in work embeds is clickable:

  • Author names → Author's AO3 profile
  • Fandoms → Fandom tag on AO3
  • Relationships/Characters/Tags → Tag search on AO3
  • Series → Series page (if ID available)

Click any blue link in the embed to explore related works on AO3!

ℹ️ Collections

Collection names are displayed as plain text (not clickable links) due to AO3 API limitations.

📜 Long Works

Works with extensive metadata (100+ tags, many relationships/characters) may split across multiple embed messages due to Discord's 25-field limit. This is normal!

🔢 Work IDs

You can use either the work ID or full URL in any command:

/workinfo work_id:12345678
/workinfo work_id:https://archiveofourown.org/works/12345678

Both work identically!
