Reading Status

/markread <work_id>
Mark a work as finished reading. Sets progress to 100% and records finish date.
Example: /markread work_id:12345678
/markreading <work_id>
Mark a work as currently reading. Records start date if not already set.
Example: /markreading work_id:12345678
/markunread <work_id>
Reset a work to unread status. Clears progress and dates.
Example: /markunread work_id:12345678
/read
Show all works you've finished reading.
/reading
Show works you're currently reading.
/unread
Show your to-read list (works not started).
📊

Progress Tracking

/progress <work_id> <chapters>
Set how many chapters you've read. Auto-completes work at 100%.
Example: /progress work_id:12345678 chapters_read:15
/progresswords <work_id> <words>
Set how many words you've read.
Example: /progresswords work_id:12345678 words_read:50000
/addchapter <work_id> [num]
Add chapters to your progress (default: 1).
Example: /addchapter work_id:12345678 chapters:3
/addwords <work_id> <words>
Add words to your progress.
Example: /addwords work_id:12345678 words:5000
/showprogress <work_id>
Display visual progress bars for chapters and words.
Example: /showprogress work_id:12345678
/inprogress
Show all works with tracked progress.
/resetprogress <work_id>
Reset progress tracking to zero.
Example: /resetprogress work_id:12345678

Ratings & Notes

/rate <work_id> <1-5>
Rate a work from 1-5 stars. Choose from dropdown menu.
Example: /rate work_id:12345678 rating:5
/unrate <work_id>
Remove your rating from a work.
Example: /unrate work_id:12345678
/toprated
View your highest rated works.
/note <work_id> <text>
Add a personal note to a work.
Example: /note work_id:12345678 note_text:Amazing ending!
/notes <work_id>
View your notes for a work.
Example: /notes work_id:12345678
/clearnote <work_id>
Delete your notes from a work.
Example: /clearnote work_id:12345678
🔖

Custom Tags

/tag <work_id> <tags>
Add custom tags (comma separated). Tags are personal to you.
Example: /tag work_id:12345678 tags:comfort fic, slow burn
/untag <work_id> <tags>
Remove custom tags from a work.
Example: /untag work_id:12345678 tags:slow burn
/tagged <tag>
Show all works with a specific custom tag.
Example: /tagged tag:comfort fic
/mytags
View all your custom tags with work counts.

Reading Lists

/addlist <list_name> <work_id>
Add a work to a list (creates list if needed).
Example: /addlist list_name:favorites work_id:12345678
/removelist <list_name> <work_id>
Remove a work from a list.
Example: /removelist list_name:favorites work_id:12345678
/showlist <list_name>
Display all works in a list.
Example: /showlist list_name:favorites
/mylists
View all your reading lists with work counts.
/deletelist <list_name>
Delete an entire reading list.
Example: /deletelist list_name:old-list

Search & Browse

/search <keyword>
Search across titles, fandoms, tags, relationships, authors, and your custom tags.
Example: /search keyword:coffee shop
/fandom <name>
Show all works in a specific fandom.
Example: /fandom name:Harry Potter
/author <name>
Show all works by a specific author.
Example: /author name:AuthorName
/series <name>
Show all works in a series.
Example: /series name:My Favorite Series
/collection <name>
Show all works in a collection.
Example: /collection name:Collection Name
/list [limit]
List all your saved works (default: 10, max: 25).
Example: /list limit:20
/recent [limit]
Show recently added works.
Example: /recent limit:15
/completed
Show only complete fics (not WIPs).
/wip
Show only works in progress.

Info & Management

/workinfo <work_id>
Display complete work information with all metadata. Fetches from AO3 if not in database.
Example: /workinfo work_id:12345678
/quickview <work_id>
Display condensed work information (summary truncated, fewer fields).
Example: /quickview work_id:12345678
/custom <work_id>
Show only YOUR personal data (status, rating, tags, notes, progress).
Example: /custom work_id:12345678
/quickadd <...>
Manually add a work without fetching from AO3. See Guide for full parameters.
Example: /quickadd work_id:12345678 title:"Title" author:"Author" fandom:"Fandom"
/delete <work_id> [force]
Delete a work from your database. Shows confirmation unless force=true.
Example: /delete work_id:12345678 or /delete work_id:12345678 force:true
/export
Download a JSON backup of all your works and lists.

Statistics

/stats
View your complete reading statistics (works, words, fandoms, etc.).
/userstats [user]
View another user's statistics (or your own if no user specified).
Example: /userstats user:@FriendName
/serverstats
View server-wide statistics including most popular works and fandoms.

Graphs & Visualizations

/readinggraph [user]
Library growth over time (line graph).
Example: /readinggraph or /readinggraph user:@FriendName
/wordsgraph [user]
Cumulative words over time (line graph).
Example: /wordsgraph user:@FriendName
/fandomgraph [user]
Top fandoms pie chart.
Example: /fandomgraph
/statusgraph [user]
Reading status breakdown (bar chart).
Example: /statusgraph
/ratingsgraph [user]
Rating distribution (bar chart).
Example: /ratingsgraph
/monthlygraph [user]
Works added per month (bar chart).
Example: /monthlygraph
/servergraph
Server-wide comparison of library sizes (horizontal bar chart).
/allgraphs [user]
Generate ALL graphs at once! Sends 6 graph images.
Example: /allgraphs or /allgraphs user:@FriendName

Recommendations

/recommend <user> <work_id> [note]
Recommend a work to another user with optional note.
Example: /recommend user:@Friend work_id:12345678 note:You'll love this!
/recs
View all your pending recommendations from others.
/acceptrec <work_id>
Accept a recommendation and add work to your library. Adds a note about who recommended it.
Example: /acceptrec work_id:12345678
/declinerec <work_id>
Decline and remove a recommendation.
Example: /declinerec work_id:12345678
/sentto [user]
View recommendations you've sent (optionally filter by user).
Example: /sentto or /sentto user:@Friend

Public Lists & Subscriptions

/makepublic <list_name>
Make one of your lists public for others to view and subscribe to.
Example: /makepublic list_name:favorites
/makeprivate <list_name>
Make a public list private again.
Example: /makeprivate list_name:favorites
/listdesc <list_name> <description>
Add a description to your public list.
Example: /listdesc list_name:favorites description:My all-time favorite fics!
/publiclists [user]
Browse all public lists (optionally filter by user).
Example: /publiclists or /publiclists user:@Someone
/viewlist <user> <list_name>
View another user's public list.
Example: /viewlist user:@Friend list_name:favorites
/subscribe <user> <list_name>
Subscribe to a public list for update notifications (when new works are added).
Example: /subscribe user:@Friend list_name:favorites
/unsubscribe <user> <list_name>
Unsubscribe from a public list.
Example: /unsubscribe user:@Friend list_name:favorites
/mysubscriptions
View all lists you're subscribed to.

Book Clubs

/clubs
View all book clubs on the server.
/clubinfo <club_name>
View detailed information about a club including current read and member progress.
Example: /clubinfo club_name:MyBookClub
/joinclub <club_name>
Join a book club.
Example: /joinclub club_name:MyBookClub
/leaveclub <club_name>
Leave a book club (owners must transfer or delete instead).
Example: /leaveclub club_name:MyBookClub
/myclubs
View all clubs you're a member of with your reading status.
/clubmembers <club_name>
View all members and their reading progress on current read.
Example: /clubmembers club_name:MyBookClub
/createclub <club_name>
Create a new book club (you become the owner).
Example: /createclub club_name:My Awesome Club
/clubdesc <club_name> <description>
Set club description (owner only).
Example: /clubdesc club_name:MyClub description:We read romance fics!
/setclubread <club_name> <work_id>
Set the current club read (owner only). Notifies all members via DM.
Example: /setclubread club_name:MyClub work_id:12345678
/acceptclubread <club_name>
Add the current club read to your library.
Example: /acceptclubread club_name:MyClub
/deleteclub <club_name>
Delete a club (owner only).
Example: /deleteclub club_name:MyClub

Settings & Help

/settings
View all your current settings and quick stats.
/recnotify <dm/channel/both>
Set how you receive recommendation notifications.
Example: /recnotify setting:dm (options: dm, channel, both)
/privacy <public/private>
Set profile privacy (reserved for future features).
Example: /privacy setting:private
/displaymode <detailed/compact>
Set default display mode for work info (reserved for future features).
Example: /displaymode setting:compact
/listlimit <5-25>
Set how many items show in lists (default: 10).
Example: /listlimit limit:15
/clubnotify <on/off>
Enable/disable book club notifications (when new club read is set).
Example: /clubnotify setting:off
/subnotify <on/off>
Enable/disable subscription notifications (when lists you follow are updated).
Example: /subnotify setting:off
/help [category]
Show help information. Categories: reading, progress, ratings, tags, lists, search, info, social, recs, shared, clubs, graphs, settings.
Example: /help or /help category:graphs
/commands
Show complete 4-page list of all commands.
