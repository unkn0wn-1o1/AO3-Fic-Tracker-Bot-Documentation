✨ Key Highlights

Automatic Work Detection

Simply paste any AO3 link in your Discord server and the bot automatically detects it, fetches all metadata (title, author, fandom, tags, summary, word count, chapter count, etc.), and saves it to your personal library. No commands needed!

Interactive AO3 Links

Every piece of metadata in work embeds is clickable and links back to AO3:

  • Author names → Author's AO3 profile
  • Fandoms → Fandom tag search on AO3
  • Relationships/Characters/Tags → Tag search pages
  • Series → Series page (when ID is available)

Private & Personal

Your library is private to you. Each user has their own data - your ratings, notes, tags, and reading progress are only visible to you unless you choose to share via public lists or recommendations.

Rich Metadata

The bot fetches everything from AO3: title, author, rating, category, fandom, relationships, characters, tags, summary, word count, chapters, completion status, series, and collections. All with clickable links back to AO3!

Works Offline

Once a work is saved, all data is cached. View your library, search, and track progress even if AO3 is down. Only first-time additions need to fetch from AO3.
